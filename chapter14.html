- 원그리기
arc(x, y, r, startAngle, endAngle [, counterClockwise])

• x, y： 원의 중점
• r： 원의 반지름
• startAngle： 원이나 호의 시작점을 라디안으로 표시합니다.
• endAngle： 원이나 호의 끝점을 라디안으로 표시합니다.
• counterClockwise： 반시계 방향으로 그릴지의 여부


const canvas = document.querySelector('canvas');
const ctx = canvas.getContext("2d");
ctx.fillStyle = "yellow";// 채우기 색상
ctx.strokestyle = "red";// 선 색상
ctx.beginPath()
ctx.arc(200, 150, 100, 0, Math.PI * 2, true);
ctx.closePath(); // 경로를 닫습니다.
ctx.fiUO; // 색상을 채웁니다.
ctx.stroke();
